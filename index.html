    How do I use a stopwatch online?

    1. When you're ready to start counting, press the Start button.

    2. Use the Lap button to check the time without stopping the timer.

    3. Press Stop to pause the stopwatch and check the end result.

    4. Now, you can either Reset and start again or continue from the current time.
